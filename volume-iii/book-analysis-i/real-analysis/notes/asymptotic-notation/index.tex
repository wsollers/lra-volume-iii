\section{Asymptotic Notation}
\label{sec:asymptotic-notation}

\begin{toolkitbox}{Toolkit: Asymptotic Notation}
Asymptotic notation records relative size near a limiting process. The atomic
notions formalised below are:
\begin{itemize}
  \item \textbf{little-o at a point} --- one function is negligible compared
        with another as the input approaches a point;
  \item \textbf{little-o at infinity} --- the same negligible comparison along
        an unbounded input range;
  \item \textbf{increment form} --- the notation $r(h)=o(h)$ as $h\to 0$;
  \item \textbf{algebra rules} --- sums and bounded factors preserve
        negligible errors;
\end{itemize}
\end{toolkitbox}
