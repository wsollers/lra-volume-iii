\begin{proposition*}[Cauchy Criterion for Function Limits]
$\lim_{x\to c}f(x)$ exists if and only if $\forall\varepsilon>0\;\exists\delta>0\;\forall x,y \in A: 0<|x-c|<\delta \wedge 0<|y-c|<\delta \Rightarrow |f(x)-f(y)|<\varepsilon$.
\end{proposition*}
