\begin{proposition*}[Quotient Undefined When the Denominator Vanishes]
Let \(A\subseteq\mathbb{R}\) and let \(f,g:A\to\mathbb{R}\).  If there is
some \(x_0\in A\) such that \(g(x_0)=0\), then the pointwise quotient
formula \((f/g)(x)=f(x)/g(x)\) does not define a function \(A\to\mathbb{R}\).
\end{proposition*}
