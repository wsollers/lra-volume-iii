\begin{theorem*}[Uniform Boundedness Transfer Under Pointwise Limit]
Let \((f_n)\) be uniformly bounded on \(A\).  If \(f_n(x)\to f(x)\) for each
\(x\in A\), then \(f\) is bounded on \(A\).
\end{theorem*}
