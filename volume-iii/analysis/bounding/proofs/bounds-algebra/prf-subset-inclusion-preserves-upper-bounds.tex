\begin{lemma*}[Subset Inclusion Preserves Upper Bounds]
Let $A \subseteq B \subseteq \mathbb{R}$. If $u$ is an upper bound of $B$, then $u$ is an upper bound of $A$.
\end{lemma*}

\begin{proof}[Professional Standard Proof]
\LRAProofBodyStart
Let $A$ and $B$ be subsets of $\mathbb{R}$ such that $A \subseteq B$. Let $u \in \mathbb{R}$ be an upper bound of $B$. By the definition of an upper bound, $b \le u$ for all $b \in B$. Since $A \subseteq B$, every element $a \in A$ is also an element of $B$. It follows that $a \le u$ for all $a \in A$. Thus, $u$ is an upper bound of $A$.
\end{proof}

\begin{proof}[Detailed Learning Proof]
\LRAProofBodyStart
\textbf{Step 1.} Identify the hypotheses and definitions.
We are given the set inclusion $A \subseteq B$ and the fact that $u$ is an upper bound for $B$. This means that for any $b$ chosen from $B$, $b \le u$.

\textbf{Step 2.} Relate elements of $A$ to the elements of $B$.
By the definition of a subset, the condition $A \subseteq B$ implies that every element $a$ belonging to $A$ must also be a member of $B$.

\textbf{Step 3.} Apply the upper bound condition to the subset.
Because every $a \in A$ is also an element of $B$, and we know $u$ is greater than or equal to every element in $B$, it must be that $a \le u$ for every $a \in A$.

\textbf{Step 4.} Conclude using the definition of an upper bound.
Since $u$ satisfies the upper bound condition for all elements in $A$, $u$ is, by definition, an upper bound for $A$.
\end{proof}

\begin{remark*}[Interpretation]
This result demonstrates the monotonicity of the upper bound property relative to set inclusion. Geometrically, if a point $u$ lies to the right of every point in a set $B$, and $A$ is entirely contained within $B$, then $u$ must also lie to the right of every point in $A$. Consequently, the set of upper bounds for $B$ is a subset of the set of upper bounds for $A$.
\end{remark*}
