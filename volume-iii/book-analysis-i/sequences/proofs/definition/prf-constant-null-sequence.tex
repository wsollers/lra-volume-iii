\begin{theorem*}[Constant Null Sequence]
Let $(x_n)$ be a real sequence, and let $c \in \mathbb{R}$. If $x_n = c$
for every $n \in \mathbb{N}$, then $(x_n)$ is a null sequence if and only if
$c = 0$.

\noindent
\end{theorem*}
