\begin{proposition*}[Union Is Bounded Below Exactly When Both Pieces Are]
For nonempty $A,B\subseteq\mathbb{R}$, $A\cup B$ is bounded below if and only
if $A$ and $B$ are bounded below.
\end{proposition*}
