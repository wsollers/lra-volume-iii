\begin{proposition*}[Pointwise Operation Evaluation]
Let \(A\subseteq\mathbb{R}\), let \(f_1,\ldots,f_n:A\to\mathbb{R}\),
and let \(\Phi:\mathbb{R}^n\to\mathbb{R}\) be an \(n\)-variable real
operation.  If \(F\) is the pointwise operation determined by
\(\Phi\) and \(f_1,\ldots,f_n\) on \(A\), then for every \(x\in A\),
\[
  F(x)
  =
  \Phi\bigl(f_1(x),\ldots,f_n(x)\bigr).
\]
\end{proposition*}
