\begin{topicbox}{Intervals as Ordered Subsets}
\begin{exposition}
Intervals are subsets described by order conditions. An open interval
\((a,b)\) consists of points strictly between \(a\) and \(b\). A closed
interval \([a,b]\) includes the endpoints as well. Half-open intervals mix
these two conventions. The notation is compact, but it always depends on the
ambient ordered system: a real interval contains real points, while a rational
interval contains only rational points unless the rationals have first been
embedded into a larger system.
\end{exposition}
\end{topicbox}
