\begin{theorem*}[Linearity of the Cauchy Integral]
If \(f\) and \(g\) are Cauchy-integrable, then the integral is linear.
\end{theorem*}
\begin{proof}[Professional Standard Proof]
\LRAProofBodyStart
For each partition, \(C(\alpha f+\beta g,P)=\alpha C(f,P)+\beta C(g,P)\).
Pass to the existing limits.
\end{proof}
\begin{proof}[Detailed Learning Proof]
\LRAProofBodyStart
Linearity is already true for finite sums. The integral is the limit of those
finite sums, so the same algebra survives the limiting process.
\end{proof}
\begin{remark*}[Proof structure]
Prove linearity at the sum level, then pass to limits.
\end{remark*}
